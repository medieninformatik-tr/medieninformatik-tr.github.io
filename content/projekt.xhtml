Aufgabe

Semesteraufgabe zu „Medieninformatik“, WS 2014/15

Entwickeln Sie eine Website, die eine Bestellung von Sportschuhen unterstützt.
Nutzen Sie dazu eine XML-Datei analog zu „transfer.tr.fh-hannover/web- home/medieninformatik/ajax/Katalog_Running.xml“ sowie die Technologien XML/XHTML, HTML5, CSS3, JQuery undAjax.
Bei Ihrer Lösung verwenden Sie in JavaScript die Anwendungsklassen „Sportschuh“ und „Laufschuh“.
Auf der Website sollen die Sport- bzw. Laufschuhe geeignet präsentiert werden.
Mit einem geeigneten Dialog soll ein Bestellformular bereitgestellt werden, über das Bestellungen per Mail abgesetzt werden können.
Verwenden Sie das Framework JQuery (siehe Datei jquery.js unter transfer.tr.fh-hannover/web-home/medieninformatik/ajax).
Richten Sie für Ihre Entwicklungsaufgabe ein Projekt in Oxygen ein und nutzen Sie XHTML.
Auflockernde Animationen können die Website bereichern.

Vorgehensweise: Bilden Sie zunächst mittels OmniGraffle ein Wireframe-Modell für ihre Website und legen Sie dann die erforderlichen HTML5-Rahmen fest.
An die HTML5- Rahmendokumente werden zur Laufzeit vordefinierte CSS3-Layoutklassen und Event- Handler analog zum Beispiel unter „beispiel_technologiemix“ gebunden.
Die Datenversorgung der Anwendungsobjekte aus dem Sportschuhkatalog soll per Ajax erfolgen.
Es sollen nur die im jeweiligen Dialog erforderlichen Daten beschafft werden.
Die Anwendungsklassen sollen analog zum Beispiel unter „beispiel_technologiemix“ realisiert werden und vor der weiteren Einbindung in einer Testumgebung getestet werden.
Zur Verwendung von Ajax arbeiten Sie bitte mit dem http-Protokoll.
